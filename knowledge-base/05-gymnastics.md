## Foundational Principle: Body Tension

All gymnastics movements are built on one foundation: **active body tension**. Without the ability to maintain a rigid, tight midline under load, no gymnastics skill can be performed safely or efficiently.

Body tension is trained through:
- Hollow body hold
- Arch body hold
- Plank variations
- L-sit progressions

**These are prerequisites.** Do not program kipping pull-ups, bar muscle-ups, or ring movements until the athlete can hold a hollow body for 30+ seconds with no compensation.

## Movement Progression Trees

### Pull-Up Tree
```
Ring row (feet elevated) → Jumping pull-up (slow negative) → 
Banded pull-up → Strict pull-up × 5 → 
Kipping swing (hollow/arch) → Kipping pull-up → 
Butterfly pull-up → Chest-to-bar pull-up → 
Bar muscle-up
```

**Prerequisite for kipping:** Strict pull-up × 3 minimum. Kipping without pulling strength leads to shoulder impingement.

### Handstand Tree
```
Box pike hold → Kick-up to wall (chest to wall) → 
Freestanding kick-up → Wall walk → 
Handstand hold (wall) 30 sec → Shoulder taps on wall → 
Freestanding handstand → Handstand walk (short distances)
```

### Handstand Push-Up Tree
```
Pike push-up → Box pike push-up (feet elevated) → 
Deficit pike push-up → Wall HSPU (kipping) → 
Strict HSPU × 3 → Deficit strict HSPU
```

### Ring Dip Tree
```
Box dip → Parallel bar dip (strict) → 
Ring support hold (stable rings) → Ring dip (kipping) → 
Strict ring dip × 5 → Weighted ring dip
```

### Muscle-Up Tree
```
Ring row + ring dip (separate, high proficiency) → 
Banded ring muscle-up → Ring muscle-up (kipping) → 
Strict ring muscle-up
```

## Gymnastics Session Structure (30 min cap)

**Block 1 — Warmup (5 min)**
- Wrist circles, shoulder CARs (Controlled Articular Rotations)
- Cat-cow, thoracic rotation
- Band pull-aparts × 20
- Active hang × 30 sec

**Block 2 — Foundational tension (5 min)**
- Hollow body hold 3 × 20–30 sec
- Arch body hold 3 × 20–30 sec
These are always present in a gymnastics session. Non-negotiable.

**Block 3 — Skill progression (15–20 min)**
- 4–6 sets of the target skill or its current progression level
- Rest fully between sets (this is skill work, not conditioning)
- Rep quality matters more than rep count
- If form breaks down, reduce the progression level immediately

**Prescription format:**
- Skill work: 5 sets × 3–5 reps (quality focus, full rest ~90 sec between sets)
- Holds: 3–5 sets × 15–30 sec (build toward longer holds)

## 4-Week Gymnastics Base Cycle

**Goal:** Develop strict pulling and pushing strength plus body tension. Target: athlete can perform 5 strict pull-ups and maintain 30-second hollow body hold by end of week 4.

| Week | Pull focus | Push focus | Tension |
|---|---|---|---|
| 1 | Ring row / banded pull-up | Box pike push-up | Hollow hold 3×20 sec |
| 2 | Banded pull-up / jumping pull-up with slow negative | Pike push-up | Hollow hold 3×25 sec |
| 3 | Strict pull-up (assisted or unassisted) | Deficit pike push-up | Hollow hold 3×30 sec |
| 4 (deload) | Strict pull-up × 3–5, no volume | Box pike push-up | Hollow hold 2×30 sec |

## Scaling Principles

Every gymnastics movement has a scale that preserves the movement pattern but reduces the load or complexity. The scale is not a failure — it is the correct place to be until the prerequisite is achieved.

**Priority:** Movement quality > movement complexity. Always regress to the highest-quality level the athlete can maintain.

**Scaling hierarchy (pull-up as example):**
1. Bar muscle-up → kipping pull-up
2. Kipping pull-up → strict pull-up
3. Strict pull-up → banded pull-up
4. Banded pull-up → ring row
5. Ring row → feet-elevated ring row
